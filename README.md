# Neural Tasks

Neural Tasks est une application web interactive de gestion de tâches au design futuriste, développée en HTML, CSS et JavaScript natif.

## Fonctionnalités

- Effets sonores interactifs (Web Audio API)
- Réorganisation des tâches par glisser-déposer (Drag & Drop)
- Indicateur de progression circulaire animé 
- Sauvegarde automatique des données via LocalStorage
- Interface moderne au style cyber-néon

## Technologies utilisées

- HTML5
- CSS3 (animations et effets visuels avancés)
- JavaScript 
- Web Audio API
- LocalStorage
